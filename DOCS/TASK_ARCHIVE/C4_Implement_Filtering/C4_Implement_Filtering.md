# Task C4: Реализовать фильтрацию (All/Active/Completed)

**Статус**: Completed ✅  
**Начало**: 2025-01-27  
**Приоритет**: High  
**Оценка**: 1 день

## Описание

Реализовать фильтрацию задач в компоненте TodoList. Пользователь должен иметь возможность переключаться между отображением всех задач, только активных или только завершенных.

## Зависимости

- [x] C3: Реализовать компонент TodoList (Completed ✅)

## Текущее состояние

В `TodoList.vue` отображаются все задачи без фильтрации. Нужно добавить:
- Кнопки/переключатели для фильтров (All, Active, Completed)
- Логику фильтрации на клиенте
- Визуальное выделение активного фильтра

## Требования

### Цель

Добавить фильтрацию задач в компонент TodoList с тремя состояниями:
- **All**: Показать все задачи
- **Active**: Показать только задачи со статусом `active`
- **Completed**: Показать только задачи со статусом `completed`

### Функциональность

1. **UI для фильтров**:
   - Три кнопки/ссылки: "All", "Active", "Completed"
   - Активный фильтр визуально выделен
   - Фильтры расположены в удобном месте (обычно под полем ввода или над списком)

2. **Логика фильтрации**:
   - Фильтрация выполняется на клиенте (не требует запросов к API)
   - При изменении фильтра список обновляется реактивно
   - Фильтр сохраняется в состоянии компонента

3. **Интеграция**:
   - Фильтрация работает с существующим списком задач
   - Обновление/удаление задач работает корректно с фильтрами

### Примечание

Фильтрация выполняется на клиенте. В будущем можно добавить фильтрацию на сервере для оптимизации, но это не входит в текущую задачу.

## План выполнения

### Шаг 1: Добавление состояния фильтра
- [ ] Создать reactive переменную `currentFilter` со значениями: 'all', 'active', 'completed'
- [ ] Установить значение по умолчанию: 'all'

### Шаг 2: Реализация логики фильтрации
- [ ] Создать computed свойство `filteredTodos` для фильтрации списка
- [ ] Реализовать логику фильтрации по статусу

### Шаг 3: UI для фильтров
- [ ] Добавить разметку для кнопок фильтров
- [ ] Реализовать обработчики кликов для переключения фильтров
- [ ] Добавить визуальное выделение активного фильтра

### Шаг 4: Стилизация
- [ ] Добавить стили для кнопок фильтров
- [ ] Добавить стили для активного состояния
- [ ] Убедиться, что фильтры хорошо видны и доступны

## Критерии приемки

✅ Фильтр "All" показывает все задачи  
✅ Фильтр "Active" показывает только активные задачи  
✅ Фильтр "Completed" показывает только завершенные задачи  
✅ Активный фильтр визуально выделен  
✅ Переключение фильтров работает мгновенно  
✅ Фильтрация работает корректно при обновлении/удалении задач
