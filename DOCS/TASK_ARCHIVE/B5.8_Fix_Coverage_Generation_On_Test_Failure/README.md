# Task B5.8: Исправить генерацию coverage при падении тестов

**Статус**: Completed ✅  
**Завершено**: 2025-01-27

## Краткое описание

Исправлена проблема, когда coverage.xml не загружался в Codecov, если тесты падали. Добавлена проверка существования файла перед загрузкой.

## Основные результаты

- ✅ Добавлена проверка существования coverage.xml перед загрузкой
- ✅ Используется `hashFiles()` для проверки файла
- ✅ Используется `always()` для выполнения даже при ошибках
- ✅ Codecov получает данные даже при ошибках тестов

## Ключевые решения

- Условие `if: always() && hashFiles('coverage.xml') != ''` для проверки файла
- Позволяет загружать coverage даже если тесты падают
- Не блокирует workflow при отсутствии файла
