## Описание

Написать E2E тест для проверки редактирования задачи через UI. Тест должен создать задачу, выполнить двойной клик на тексте задачи, отредактировать текст, нажать Enter для сохранения и проверить результат.

## Зависимости

- [x] D3.1: Настроить Laravel Dusk для E2E тестов (Completed ✅)
- [x] C6: Реализовать редактирование задачи (Completed ✅)

## Текущее состояние

Тесты для создания задачи существуют. Нужно создать тест для редактирования через double-click.

## Требования

### Цель

Создать E2E тест, который:
- Создает задачу
- Выполняет двойной клик на тексте задачи
- Редактирует текст в поле ввода
- Нажимает Enter для сохранения
- Проверяет, что текст обновился

### Функциональность

1. **Создание задачи**:
   - Создать задачу для редактирования

2. **Редактирование**:
   - Найти текст задачи
   - Выполнить двойной клик
   - Проверить, что появилось поле ввода
   - Ввести новый текст
   - Нажать Enter

3. **Проверка**:
   - Проверить, что текст задачи обновился
   - Проверить, что поле редактирования закрылось
