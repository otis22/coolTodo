**Статус**: Completed ✅

**Оценка**: 0.5 дня

## Описание

Реализовать счетчик активных (незавершенных) задач. Счетчик должен отображаться в интерфейсе и обновляться при изменении статуса задач.

## Зависимости

- [x] C3: Реализовать компонент TodoList (Completed ✅)
- [x] C4: Реализовать фильтрацию (Completed ✅)

## Текущее состояние

В `TodoList.vue` есть список задач, но нет счетчика активных задач. Нужно добавить:
- Computed свойство для подсчета активных задач
- UI элемент для отображения счетчика
- Автоматическое обновление при изменении статуса задач

## Требования

### Цель

Добавить счетчик активных задач, который:
- Показывает количество задач со статусом `active`
- Обновляется автоматически при изменении статуса задач
- Отображается в удобном месте интерфейса

### Функциональность

1. **Подсчет активных задач**:
   - Использовать computed свойство для реактивного подсчета
   - Фильтровать задачи по статусу `active`

2. **Отображение**:
   - Показывать счетчик в формате: "X items left" или "X активных задач"
   - Разместить счетчик в удобном месте (обычно слева от фильтров или над списком)

3. **Реактивность**:
   - Счетчик обновляется при изменении статуса задачи
   - Счетчик обновляется при добавлении/удалении задач
