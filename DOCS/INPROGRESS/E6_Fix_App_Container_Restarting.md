## Описание

App контейнер постоянно перезапускается. В логах видно, что он запускает интерактивную оболочку PHP (`php > Interactive shell`) вместо работы как сервис.

## Проблема

**Текущая ситуация**:
- Контейнер `cooltodo_app` в статусе `Restarting`
- В логах: `php > Interactive shell`
- Контейнер не может работать как сервис для PHP-FPM

**Причина**:
- Dockerfile.dev, вероятно, не настроен правильно для работы с PHP-FPM
- Или команда запуска контейнера не указана
