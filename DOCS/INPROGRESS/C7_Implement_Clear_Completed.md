# Task C7: Реализовать кнопку "Clear completed"

**Статус**: In Progress  
**Начало**: 2025-01-27  
**Приоритет**: High  
**Оценка**: 0.5 дня

## Описание

Реализовать кнопку "Clear completed", которая удаляет все завершенные задачи одним кликом.

## Зависимости

- [x] C3: Реализовать компонент TodoList (Completed ✅)
- [x] C1: Создать API сервис (Completed ✅)

## Текущее состояние

В `TodoList.vue` есть список задач и фильтры, но нет кнопки для массового удаления завершенных задач. Нужно добавить:
- Кнопку "Clear completed"
- Логику подсчета completed задач
- Интеграцию с API для удаления всех completed задач
- Условное отображение (показывать только если есть completed задачи)

## Требования

### Цель

Добавить кнопку "Clear completed", которая:
- Удаляет все задачи со статусом `completed`
- Отображается только когда есть completed задачи
- Расположена в удобном месте (обычно в футере рядом со счетчиком)

### Функциональность

1. **Подсчет completed задач**:
   - Computed свойство для подсчета completed задач
   - Используется для условного отображения кнопки

2. **Удаление**:
   - При клике вызывается API метод `deleteCompleted`
   - После успешного удаления список обновляется
   - Обработка ошибок

3. **UI**:
   - Кнопка отображается только если есть completed задачи
   - Стилизация в соответствии с дизайном

## План выполнения

- [ ] Обновить список после удаления

- [ ] Добавить кнопку в футер

### Шаг 4: Стилизация
- [ ] Добавить стили для кнопки
- [ ] Убедиться, что кнопка хорошо видна и доступна

## Критерии приемки

✅ Кнопка "Clear completed" отображается только когда есть completed задачи  
✅ Кнопка удаляет все completed задачи через API  
✅ После удаления список обновляется
