# Task E7: Исправить 404 ошибки в nginx для API

**Статус**: Open  
**Приоритет**: High  
**Оценка**: 0.25 дня

## Описание

Nginx возвращает 404 для всех API запросов. Конфигурация nginx указывает на неправильный путь к Laravel приложению.

## Проблема

**Текущая ситуация**:
- Nginx конфигурация: `root /var/www/html/public;`
- Docker-compose монтирует: `./backend:/var/www/project/backend`
- Несоответствие путей приводит к 404 ошибкам

**Ошибка**:
```
GET http://localhost:8080/api/todos
→ 404 Not Found
```
