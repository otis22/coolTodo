# Task E10: Удалить устаревший атрибут version из docker-compose.yml

**Статус**: Open  
**Приоритет**: Low  
**Оценка**: 0.05 дня (5 минут)

## Описание

Docker Compose выдает предупреждение о том, что атрибут `version` устарел и будет проигнорирован. Нужно удалить его из `docker-compose.yml`.

## Проблема

**Текущая ситуация**:
- В `docker-compose.yml` есть строка `version: '3.8'`
- Docker Compose выдает предупреждение: `the attribute 'version' is obsolete, it will be ignored, please remove it to avoid potential confusion`
- Это не критическая проблема, но создает лишний шум в логах

**Причина**:
- Начиная с Docker Compose версии 2.0, атрибут `version` больше не требуется и игнорируется
- Современные версии Docker Compose определяют формат автоматически
